What is this dataset?

Standard gravity model distance calculations anchor bilateral distances at arbitrary reference points — the capital city, the most-populated city, or the simple geographic centroid. For large, economically heterogeneous countries (US, China, Brazil, Russia, India), these choices can differ by hundreds or thousands of kilometers from where economic activity is actually concentrated.

This dataset provides two theoretically motivated alternatives for every country in the sample, computed annually from satellite and gridded population data:

  • VIIRS-weighted centroid — luminosity-weighted mean center from NASA/NOAA VIIRS DNB nighttime lights. Proxies the spatial concentration of economic production.
  • LandScan-weighted centroid — population-weighted mean center from ORNL LandScan Global. Captures where people (and therefore demand) are located.

The asymmetric cross-weighting — origin by VIIRS, destination by LandScan — is theoretically motivated: exporters ship from where production is concentrated; importers receive at population centers. Applied to BACI-CEPII data via PPML, this measure yields a distance elasticity of −0.734 versus −0.625 for simple population-city distance (N = 122,528 dyad-year observations).

Usage Example

Stata — merge into BACI + CEPII gravity

// Load centroids
use "centroids_viirs_ls.dta", clear

// Merge origin VIIRS centroid
rename iso3 iso3_o
rename viirs_lon viirs_lon_o
rename viirs_lat viirs_lat_o
merge m:1 iso3_o year using baci_cepii, keep(match)

// Merge destination LandScan centroid
rename iso3_o iso3_d
... // same pattern for destination

// Compute Haversine distance (km)
// — or use the pre-computed viirs_to_ls_km field directly
gen ln_dist_vl = ln(viirs_to_ls_km)

// PPML estimation
ppmlhdfe trade ln_dist_vl contiguity comlang_off col_ever rta, ///
    absorb(iso3_o#year iso3_d#year) cluster(iso3_o#iso3_d)

Python — quick distance check

import pandas as pd
from math import radians, sin, cos, sqrt, atan2

df = pd.read_csv("centroids_viirs_ls.csv")

def haversine(lat1, lon1, lat2, lon2):
    R = 6371
    dlat = radians(lat2 - lat1)
    dlon = radians(lon2 - lon1)
    a = sin(dlat/2)**2 + cos(radians(lat1))*cos(radians(lat2))*sin(dlon/2)**2
    return 2 * R * atan2(sqrt(a), sqrt(1-a))

# Example: US VIIRS centroid → Germany LandScan centroid, 2017
usa = df[(df.iso3=="USA") & (df.year==2017)].iloc[0]
deu = df[(df.iso3=="DEU") & (df.year==2017)].iloc[0]
dist = haversine(usa.viirs_lat, usa.viirs_lon, deu.ls_lat, deu.ls_lon)
print(f"US→DE distance: {dist:.0f} km")
